### 👋 About me

CS student at **Bauman Moscow State Technical University** (CAD Systems dept).

- 🔭 **Working on:** REST API with JWT auth, Dockerized PostgreSQL backend
- 🌱 **Learning:** backend architecture, system design, SwiftUI
- 🧩 **Comfortable across:** Python, Swift, SQL
